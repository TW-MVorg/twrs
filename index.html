[SYSTEM] 專案概述 / PROJECT OVERVIEW

「台灣西部縱貫鐵路模擬器」是一款以台灣西部縱貫線為核心、強調環境與地方特色還原的鐵道模擬體驗。玩家可操作車輛、體驗站務與營運規劃，或在開放伺服器與其他玩家共同建構與維護路網。

:: 模擬範圍與目標 ::
本專案涵蓋西部幹線從基隆到高雄的完整路線，目標是以高精度還原沿線超過 **100 個車站**的建築與景觀細節。透過開放世界設計，我們打造了一個動態的世界系統，天氣與時間的變化將影響你的行駛體驗。

:: 營運與合作 ::
除了駕駛火車，玩家還能參與排班、調度、執行客運或貨運任務。在多人連線模式下，玩家社群的合作與分工是維持整個西部路網順暢運行的關鍵要素。

[DATA] 核心特色 / CORE FEATURES

  • 真實還原地形與車站風貌： 沿線地景、橋梁、沿海與山城場景皆重視在地細節，提供身臨其境的鐵道旅行感受。
  • 多樣車種與駕駛模式： 支援客運、貨運、區間車與電聯車等多種列車操作。每種車輛都具備獨特的物理模型和駕駛室介面。
  • 高級信號與安全系統： 模擬真實的台鐵信號系統、號誌規則、以及列車自動保護系統（ATP）。要求玩家嚴格遵守速度限制和進路指示，挑戰專業級的駕駛精準度。
  • 動態天氣與日夜循環： 遊戲內建 24 小時動態時間系統和多變天氣效果（如暴雨、濃霧、夕陽），這些環境因素將直接影響軌道抓地力與駕駛視野，大幅提升操作難度與真實感。
  • 生涯模式與緊急事件： 提供從入門到專家級的排程運行任務，以及隨機發生的緊急狀況（如路障、設備故障），考驗玩家的應變能力，解鎖新的職位與列車。
  • **社群協作與活動：** 定期舉辦試運轉、RP（角色扮演）活動、競速挑戰，並提供強大的伺服器社群工具支援玩家自發性的營運與路網維護。
