[SYSTEM] 專案概述 / PROJECT OVERVIEW

「台灣西部縱貫鐵路模擬器」是一款以台灣西部縱貫線為核心、強調環境與地方特色還原的鐵道模擬體驗。玩家可操作車輛、體驗站務與營運規劃，或在開放伺服器與其他玩家共同建構與維護路網。

[DATA] 核心特色 / CORE FEATURES

  • 真實還原地形與車站風貌：沿線地景、橋梁、沿海與山城場景皆重視在地細節。
  • 多樣車種與駕駛模式：支援客運、貨運、區間車與電聯車等多種列車操作。
  • 社群協作與活動：定期活動、試運轉、RP（角色扮演）活動與伺服器社群支援。

[CONFIG] 系統需求 / SYSTEM CONFIGURATION

建議配置（PC/遊戲平台）：

  • **作業系統：** Windows 10 64-bit / 或平台相容需求
  • **處理器：** 四核心或以上
  • **記憶體：** 8GB RAM（建議 16GB）
  • **顯示卡：** 支援 DirectX 11 的獨立顯示卡
  • **儲存空間：** 至少 10GB 可用空間
